FGU CHARACTER SHEET GENERATOR V2 - DEPLOYMENT SUMMARY
================================================================================

WHAT IS V2?
━━━━━━━━━━
A web-based version of the FGU character sheet generator that can be deployed
to the cloud (Vercel, Heroku, etc.) and accessed from any browser.

KEY DIFFERENCES FROM V1:
━━━━━━━━━━━━━━━━━━━━━━

V1 (Desktop):
  • GUI application (tkinter)
  • Requires Python installation
  • Local file system access
  • Single user at a time

V2 (Web):
  • Flask web application
  • No Python needed for users
  • Cloud deployment ready
  • Multiple concurrent users
  • Works on mobile, tablet, desktop

ARCHITECTURE
================================================================================

User's Browser
       ↓
   Upload Form (HTML/JS)
       ↓
  Flask Routes (/generate)
       ↓
  XML Parsing Engine
       ↓
  HTML Generator
       ↓

================================================================================
TROUBLESHOOTING DEPLOYMENT
================================================================================

Build Fails:
  • Check requirements.txt has all dependencies
  • Verify vercel.json syntax (use JSON validator)
  • Check proyectocsFGU_V2.py for syntax errors
  • Review Vercel build logs

App Shows Error on Upload:
  • File must be .xml format
  • File must be valid FGU export
  • Try with different character file
  • Check browser console (F12 > Console)

Generated HTML Looks Wrong:
  • Clear browser cache
  • Try different browser
  • Test locally first
  • Check if original XML is valid

Can't Connect to App:
  • Verify URL is correct
  • Check Vercel deployment status
  • Wait 1-2 minutes (first load is slow)
  • Try from different network
